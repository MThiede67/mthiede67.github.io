# Kerry's Fine Painting Website

A simple static website scaffold for Kerry's Fine Painting. This project includes a homepage, services, gallery, and contact section.

## Files
- `index.html` — main website structure
- `styles.css` — responsive styling
- `script.js` — mobile menu and footer year

## Local Development
1. Open this folder in VS Code.
2. Open `index.html` in the browser, or use a live server extension.
